# ctrlquery-site

Landing page and privacy policy for the [CtrlQuery](https://github.com/Garrett2558/CtrlQuery) Chrome extension.

Hosted on GitHub Pages. Static HTML + Tailwind CDN — no build step.

## Local preview

Open `index.html` directly in a browser, or run any static server from this folder:

```bash
python -m http.server 8000
# then visit http://localhost:8000
```

## Deploy

Any push to `main` auto-deploys to GitHub Pages.
